# Tutorials

Welcome to the OmniQ tutorials! These hands-on guides will help you get started with OmniQ step by step.

## Available Tutorials

- [Async Quickstart](quickstart.md) - Get started with async task queues in minutes
- [Scheduling (ETA & Interval)](scheduling.md) - Schedule tasks for delayed and recurring execution
- [Retries & Task Errors](retries.md) - Handle failures gracefully with automatic retries
- [Logging & Sync API](logging.md) - Configure logging and use the synchronous API

Each tutorial includes complete, runnable code examples that you can copy and modify.

## Prerequisites

- Python 3.10+
- Basic understanding of async/await (for async tutorials)
- OmniQ installed (`pip install omniq`)

## Getting Help

If you run into issues, check the [How-to Guides](../how-to/index.md) for solutions to common problems or the [Explanation](../explanation/index.md) section for deeper understanding.
