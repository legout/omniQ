# How-to Guides

Practical guides to help you accomplish specific tasks with OmniQ.

## Available Guides

- [Choose Backend](choose-backend.md) - Select the right storage backend for your use case
- [Run Workers](run-workers.md) - Start and manage worker processes
- [Configure Retries](configure-retries.md) - Set up retry policies and timeouts
- [Inspect Errors](inspect-errors.md) - Debug and monitor task failures
- [Configure Logging](configure-logging.md) - Customize logging output and levels

## When to Use How-to Guides

How-to guides are goal-oriented. Use them when you need to:

- Solve a specific problem
- Complete a particular task
- Find the right configuration for your needs

For learning OmniQ from scratch, start with the [Tutorials](../tutorials/index.md). For understanding why things work the way they do, see the [Explanation](../explanation/index.md) section.
